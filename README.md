# mempalace-policy

**MemPalace AutoPost** のプライバシーポリシー・利用規約を公開するための静的サイト。
Static policy site for *MemPalace AutoPost* (privacy policy & terms of service).

## ページ構成 / Pages

| ファイル | 内容 |
|---|---|
| `index.html` | ランディング（MemPalace AutoPost 概要） |
| `privacy.html` | プライバシーポリシー / Privacy Policy |
| `terms.html` | 利用規約 / Terms of Service |
| `tiktok*.txt` | TikTok ドメイン所有確認ファイル（削除しないこと） |

## プレビュー / Preview

`index.html` をブラウザで直接開く。バックエンド不要・ビルド不要の静的 HTML。

## デプロイ / Deploy

静的ホスティング（Cloudflare Pages 等）に配置するだけ。
TikTok 検証ファイルは公開ルート直下に残す必要があるため削除しない。

## ライセンス / License

ポリシー文書のため再利用は想定していません。内容に関する問い合わせは MemPalace AutoPost の窓口へ。
